**SQL Data Cleaning Project — Layoffs Dataset**

**Project Overview**
This project focuses on cleaning and preparing a real‑world layoffs dataset using SQL.
The goal was to transform messy, inconsistent raw data into a clean, analysis‑ready dataset by applying industry‑standard data cleaning techniques.

This project demonstrates my ability to:
Identify and remove duplicates
Standardize inconsistent values
Handle nulls and blank fields
Correct data types
Remove irrelevant rows and columns
Apply SQL best practices for data quality

**Tools & Technologies**
MySQL Workbench
SQL (CTEs, Window Functions, Joins, Aggregations)
CSV dataset

**Final Output**
The final cleaned dataset (layoffs2) is:
Free of duplicates
Standardized
Consistent
Properly typed
Ready for analysis or visualization
